# AutoFish Spin

Скрипт для автоматизации ловли спиннингом в Russian Fishing 4.

## Установка

Запустите `AutoFish.exe` (скомпилированный exe-файл, .NET Framework).

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| **F3**  | Пуск     |
| **F4**  | Стоп     |
| **F5**  | Запомнить эталон |
| **F6**  | Скан     |

## Режимы работы

Выбираются в выпадающем списке **«Режим»**.

### 1. Яркость
Детектирует появление текста «Снасть готова к забросу» по яркости пикселей. Если в области захвата >5% ярких пикселей 3 раза подряд — останавливает мотку и делает заброс.

Настройки:
- **Порог яркости** — минимальная яркость пикселя (0-255), который считается «ярким». Чем выше значение, тем светлее должен быть пиксель.
- **Область захвата** — X, Y, W, H координаты области на экране, где появляется текст.

### 2. Эталон
Сравнивает текущую область с запомненным эталоном (попиксельный узор яркости). Если совпадение >85% 3 раза подряд — останавливает мотку и делает заброс.

Как использовать:
1. Дождитесь появления текста «Снасть готова к забросу» в игре
2. Нажмите **«Запомнить(F5)»** — скрипт запомнит область как эталон
3. Запустите рыбалку (F3)
4. Во время мотки скрипт будет сравнивать область с эталоном и остановится при совпадении

Меньше ложных срабатываний, чем режим «Яркость», так как сравнивается именно узор пикселей.

### 3. Ручной
Мотает удочку ровно столько секунд, сколько указано в поле **«Мотка (с)»**, без захвата экрана и детекции.

Просто таймер — нажимает заброс, ждёт указанное время, мотает указанное время, снова заброс.

## Настройки

### Общие
- **Ожидание (с)** — время ожидания поклёвки перед началом мотки.
- **Мотка (с)** — время мотки (для режима «Ручной» — точное время мотки; для «Яркость» и «Эталон» — максимальное время мотки, после которого принудительный заброс).

### Координаты области захвата
**X, Y, W, H** — положение и размер области на экране, где отображается текст «Снасть готова к забросу».

Для каждого режима сохраняются свои координаты. При переключении режима настройки подгружаются автоматически.

### Кнопка «Скан» (F6)
Делает скриншот указанной области и показывает:
- в режиме **«Яркость»** — количество ярких пикселей и их процент
- в режиме **«Эталон»** — процент совпадения с эталоном
- в режиме **«Ручной»** — сообщение что скан не используется

Скриншот сохраняется в `autofish_scan.png` рядом с exe-файлом.

## Принцип работы

1. Заброс: зажимается левая кнопка мыши на 1 секунду, затем отпускается
2. Ожидание указанное время
3. Мотка: зажимается левая кнопка мыши
4. Детекция (для режимов «Яркость» и «Эталон»): каждые 500 мс проверяется область захвата
5. При срабатывании детекции или по таймауту: ЛКМ отпускается, цикл повторяется
